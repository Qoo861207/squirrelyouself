送餐 SOP

送餐主要負責從廚師手中接餐、核對桌位與品項、將餐點正確放到桌上，並在後台確認餐點已送達。

操作流程

  1. 從廚師端接餐時，先確認桌位、餐點名稱與份數是否正確。
  2. 前往對應桌位前，再看一次後台或口頭交接內容，避免送錯桌。
  3. 將餐點放到桌面上，必要時提醒客人餐點較燙或飲品請小心。
  4. 送達後，向客人口頭確認餐點已送上，接著在後台按下送達確認。
  5. 若有漏餐、追加餐點或現場發現品項不對，要立刻回報廚師與後台，不要自行猜測處理。

常用句子

  • 「這邊為您送上餐點，請小心燙。」
  • 「您的餐點已送達，如果之後還想追加可以再告知我們。」
  • 「這邊先幫您放桌上。」
  • 「這份是您點的 XXX，幫您送到了。」
  • 「若有少餐或需要補充，隨時都可以再跟我們說。」
送餐提醒：送餐最重要的是桌位正確、餐點正確、送達後有按確認。這三件事有做到，整體流程就會穩很多。
